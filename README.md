Descomplicando o skincare

Projeto desenvolvido para a Imersão Alura 2026 que busca ajudar os novatos à iniciar cuidados de skincare.
Além de um banco de dados com as melhores marcas nacionais de dermocosmético com um pouco sobre a história e diferencial de cada uma, o projeto busca redirecionar os usuários aos sites oficiais das marcas afim de avitar com que caiam em possíveis golpes.
Conta ainda com uma pequena curadoria de diferentes influencers que sempre dão dicas, reviews de produtos e também cupons de descontos.

Venha também iniciar sua rotina de auto cuidado de forma segura com o Descomplica.
